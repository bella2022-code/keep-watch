## 4.5 Trigger Algorithm

```
On focus completion:
1. Determine current scene → load that character's pool
2. 70% chance: random from full pool
3. 20% chance: weighted by time of day
   - Morning → favor group A (affirmation)
   - Afternoon → favor group B (closing)
   - Evening → favor group C (spacious)
4. 10% chance: conditional line
   - Short/long session → 100% trigger matching line
   - Cumulative ≥ 5 days → unlock C-14
5. Milestone: forced display on threshold hit (overrides all)
6. Anti-repeat: last shown is excluded from next pool
7. 5% suffix append (independent roll)
```
