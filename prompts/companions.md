```
Tiny pixel water bear (tardigrade), 4x4 px on 32x40 canvas,
translucent grey-white #ECEEF2 body, faint green bioluminescent dot #7B9268,
8-legged slow walking animation (4 frames),
zero-gravity floating slow curve path animation,
extremely small and unobtrusive
```

## Companion Dialog Bubble

```
A small pixel speech bubble, 20x12 px,
warm white #F4EFE6 background,
1px outline #2A2128,
small triangular pointer at bottom-left or bottom-right (4x2 px),
empty interior for text overlay,
rustic hand-drawn feel
```

Used to display: *squeak*, *blub*, etc. (text rendered separately in pixel font)
